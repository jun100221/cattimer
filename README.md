# CatTimer

保護猫のプロフィールとケア履歴を記録する、スマホ向けWebアプリです。

Firebaseやログイン機能は使わず、データは利用者本人のブラウザ内に保存されます。
猫の写真、プロフィール、ケア履歴、メモ、ホーム表示設定は端末ごとに独立します。

## 主な機能

- 複数の猫のプロフィール登録
- 写真の正方形トリミングと自動圧縮
- 猫ごとの記録ボタン作成
- ケア履歴とメモの保存
- アーカイブ機能
- ホーム画面表示のカスタマイズ
- バックアップの書き出し・読み込み

## iPhoneでの使い方

CatTimerは、App StoreからダウンロードするiPhoneアプリではなく、Safariからホーム画面に追加して使うWebアプリ、いわゆるPWAです。

1. iPhoneのSafariでCatTimerのURLを開きます。
2. 共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。
4. ホーム画面のCatTimerアイコンから起動します。

一度読み込んだあとであれば、基本画面はオフラインでも開けます。猫の登録情報とケア履歴は、利用している端末のブラウザ内に保存されます。

端末を変えたり、ブラウザのデータを削除したりすると記録は引き継がれません。

## 重要な注意

CatTimerは、Firebaseやサーバーにデータを保存しません。
記録は利用者本人のiPhone、iPad、Macなどのブラウザ内に保存されます。

そのため、Safariの「履歴とWebサイトデータを消去」を行うと、CatTimerの記録も消える可能性があります。
ホーム画面のCatTimerを削除した場合や、端末の初期化・故障時にも記録を失う可能性があります。

大切な記録は、定期的にバックアップを書き出してください。

## バックアップ

大切な記録は、登録画面の「バックアップ」からJSONファイルを書き出し、iCloud Driveやファイルアプリなどに保存してください。
復元するときは、同じ画面からバックアップファイルを読み込めます。

## 公開時の注意

iPhoneでPWAとして安定して使うには、HTTPSで配信してください。GitHub Pages、Netlify、Vercel、Cloudflare Pagesなどに置くと、Safariからホーム画面に追加して使えます。

GitHub Pagesで公開する場合、このフォルダの内容をリポジトリに置き、Pagesを有効にしてください。
